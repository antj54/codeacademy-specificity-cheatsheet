CSS Specificity

It Doesn't Have to Be Scary

Every CSS selector has its place in the specificity hierarchy. There are four categories which define the specificity level of a selector.
See the table below.

Specificity Level Selector Example
Highest Inline Styles <h1 style="color: pink;">
Second IDs #navbar
Third Classes .test, :hover
Lowest Elements h1
